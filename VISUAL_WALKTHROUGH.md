## WHAT IF SOMETHING LOOKS WRONG?

### Blank Page?
1. Hard refresh: `Ctrl+Shift+R`
2. Wait 3 seconds
3. Should load

### Page looks weird?
1. Hard refresh: `Ctrl+Shift+R`
2. Check zoom level: `Ctrl+0`
3. Try different browser

### No data showing?
- This is normal in dev mode
- Database is mocked
- Data won't persist on refresh

# POS Application

A multi-purpose POS application for Supermarkets, Installment Sales Outlets, and Warehouses.

## Features

-   Elegant Login Screen
-   Main Dashboard with daily and monthly revenue summary
-   POS (Invoice Issuing) Screen
-   Product Management
-   Customers and Suppliers Management
-   Expenses Management
-   Reports
-   Settings (Language, Currency, Logo, Printer)
-   Users and Permissions
-   Categories
-   Returns (Refunds)
-   Dark/Light mode
-   Backup and restore to Google Drive

## Technologies

-   React
-   Tailwind CSS
-   TypeScript
-   SQLite (better-sqlite3)
-   i18next
-   Context API
-   Framer Motion
-   PrintJS
-   jsPDF
-   Electron
-   Recharts
-   xlsx
